# emowowo - 儿童动画编剧工具

为1-5岁儿童动画片批量创作独立剧情脚本和分镜脚本。

## 核心功能

- 批量生成多集独立完整的动画剧情
- 每集围绕统一主题，但剧情独立
- 支持自定义剧情结构
- 生成分镜脚本（三种详细程度）
- 融入正向价值观
- 突出 Team Work

## 使用流程

### Step 1: 收集用户输入

向用户收集以下参数：

| 参数 | 说明 | 示例 |
|------|------|------|
| 角色list | 纯文本格式的角色描述 | 小兔子：5岁，活泼可爱，擅长... |
| 主题 | 故事大方向 | 三岁孩子适应幼儿园生活 |
| 集数 | 要创作的集数 | 5集 / 10集 / 30集 |
| 每集时长 | 用户设定 | 3分钟 / 5分钟 |
| 分镜详细程度 | 简洁 / 中等 / 详细 | 用户选择 |
| 每集标准结构 | 剧情结构模板 | problem→rescue→happy ending |

**每集标准结构说明**：
- 用户可自定义剧情结构模板
- 默认值：`problem→rescue→happy ending`
- 其他示例：`起因→经过→结果`、`挑战→成长→收获`、`问题→尝试→解决→欢乐`

### Step 2: 分析角色

- 理解每个角色的：年龄、性格、特长、外貌特征
- 提取角色的独特技能，用于 Team Work 设计
- 确保每个角色在故事中都有展现价值的机会

### Step 3: 批量生成剧情

**整体原则**：
- 每集是独立完整的故事
- 整体围绕同一主题展开
- 30集应有成长线或难度递进
- 每集融入1-2个正向价值观

**每集生成流程**：
1. 确定本集的 problem（问题/困难）
2. 设计 rescue（营救/解决）过程，突出 Team Work
3. 结尾 happy ending（快乐结局）
4. 融入价值观

**正向价值观库**：
- 勇气、善良、分享、诚实、友爱
- 耐心、感恩、环保、帮助他人
- 勇敢面对困难、团结合作、相互理解
- 接受不完美、珍惜友谊、保护小动物

### Step 4: 生成分镜脚本

根据用户选择的详细程度输出：

#### 简洁版
```
场景：场景名称
动作：角色关键动作
对话：角色对话内容
```

#### 中等版
```
场景：场景名称
场景描述：简短场景描绘
动作：角色动作
对话：角色对话
景别：镜头景别建议
```

#### 详细版
```
场景：场景名称
视觉氛围：色彩、光线、整体氛围
动作：角色动作描述
对话：角色对话
景别：镜头景别
时长：建议时长
音效/音乐：背景音效或音乐建议
情绪：角色情绪提示
镜头运动：镜头运动方式
```

## 分镜详细程度定义

| 等级 | 包含内容 |
|------|----------|
| **简洁** | 场景 + 关键动作 + 对话 |
| **中等** | 场景 + 描述 + 动作 + 对话 + 景别 |
| **详细** | 场景 + 视觉氛围 + 动作 + 对话 + 景别 + 时长 + 音效 + 情绪 + 镜头运动 |

## Team Work 设计原则

- 每集根据剧情需要，分配不同角色组合
- 每个角色发挥独特技能
- 强调"团结力量大"
- 避免只有一个角色解决问题
- 展现角色间的互补合作

## 幼儿内容创作要点

### 适合1-5岁儿童的特点
- 情节简单明了
- 角色动机清晰
- 冲突温和不恐怖
- 解决方式积极正向
- 画面明亮欢快
- 对话简短易懂

### 避免的内容
- 暴力、恐惧、黑暗元素
- 复杂的道德困境
- 过于抽象的概念
- 长时间紧张压抑

### 趣味性设计
- 夸张可爱的角色反应
- 意想不到的惊喜
- 朗朗上口的简短对话
- 反复出现的趣味元素

## 使用示例

### 示例输入

```
角色list：
- 小兔子毛毛：5岁，活泼开朗，擅长跳舞，乐观积极
- 小熊笨笨：4岁，力气大但有点害羞，擅长搬运
- 小狐狸聪聪：5岁，聪明机灵，擅长动脑筋解决问题
- 小猫花花：3岁，温柔可爱，擅长安慰朋友

主题：幼儿园生活
集数：5集
每集时长：3分钟
分镜详细程度：中等
每集标准结构：problem→rescue→happy ending
```

## 注意事项

1. **角色一致性**：确保每集的角色性格和行为与角色list保持一致
2. **节奏把控**：根据每集时长合理分配场景数量
3. **价值观自然融入**：避免说教，让价值观通过故事自然体现
4. **Team Work突出**：每集都要展现团队合作的力量
5. **幼儿视角**：始终用1-5岁儿童的视角看待问题和世界
