# 分镜表模板

## 分镜表通用模板

```
集数：[1/2/3]
镜头序号：[1, 2, 3...]
角色：[男1/女1/男2/女2/男3/女3/群演]
景别：[远景/全景/中景/近景/特写/大特写]
动作：[详细动作描述]
构图：[三分法/中心对称/对角线/黄金分割/框架式]
光影：[侧光/顶光/底光/逆光/柔光/硬光]
氛围：[紧张/神秘/温馨/恐惧/浪漫/悬疑]
时长：[X秒]
```

## 分镜设计原则总结

### 3秒完播率原则

1. **第1个镜头必须抓眼球**：
   - 极端特写（眼睛、瞳孔）
   - 强烈对比（明暗、色彩）
   - 突然变化（爆炸、惊吓）

2. **前3个镜头建立张力**：
   - 镜头1：极致视觉冲击
   - 镜头2：建立场景冲突
   - 镜头3：引入核心悬念

### 悬念设计原则

1. **每集结尾的悬念**：
   - 第1集结尾：神秘追踪者
   - 第2集结尾：男1生死未卜 + 男2神秘态度
   - 第3集结尾：立意升华（非悬念，而是哲学思考）

2. **悬念类型**：
   - 信息悬念："有人在追踪我"
   - 生死悬念："他还能活多久？"
   - 真相悬念："你永远想不到"

### 审美原则

1. **构图统一**：
   - 同一场景保持构图风格一致
   - 使用3-4种构图方式交替

2. **光影统一**：
   - 遵循影调设定
   - 对比光、柔和光交替

3. **节奏控制**：
   - 张弛有度，不要一直紧张
   - 快镜头与慢镜头交替
   - 时长：0.5秒（极端）- 3秒（铺垫）

### 角色标识原则

1. **标识符统一**：
   - 男1、女1、男2、女2...
   - 不使用具体描述

2. **动作描述清晰**：
   - 具体动作，如"瞳孔放大"
   - 避免模糊描述，如"感到害怕"

3. **时长合理分配**：
   - 总时长：1分钟（约60秒）
   - 镜头数量：15-18个
   - 平均时长：3-4秒/镜头
