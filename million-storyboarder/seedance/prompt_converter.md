# Seedance提示词转换规则

## 转换原则

### 1. 角色标识转换

**规则**：
- 将所有具体角色描述转换为角色标识符
- 移除外貌、服装等详细描述（已在角色库中）
- 使用标准标识符：男1、女1、男2、女2、男3、女3、群演

**示例**：
```
原始："一个戴眼镜、穿着白大褂的年轻研究员"
转换："男1"

原始："一个蓝色全息投影形式的AI女性"
转换："女1"

原始："一个穿西装、头发花白的所长"
转换："男2"
```

**多角色处理**：
- 多个角色在同一镜头：用"+"连接
- 示例："男1+女1" 表示男1和女1在同一画面

### 2. 景别描述标准化

**规则**：
- 统一使用标准景别术语
- 移除冗余描述，只保留核心景别

**景别对照表**：

| 原始描述 | 标准术语 |
|----------|----------|
| 极致特写、眼睛特写 | 大特写 |
| 特写、面部特写 | 特写 |
| 近景、上半身 | 近景 |
| 中景、腰部以上 | 中景 |
| 全景、全身及环境 | 全景 |
| 远景、广阔场景 | 远景 |
| 广角、超广角 | 远景 |

**示例**：
```
原始："瞳孔放大的极致特写"
转换："大特写"

原始："展示房间角落的全景"
转换："全景"

原始："从远处的角度拍摄的广阔场景"
转换："远景"
```

### 3. 动作描述精简化

**规则**：
- 保留核心动作，移除冗余修饰
- 使用简洁动词，避免长句
- 保留必要的动作细节（如"瞳孔放大"）

**示例**：
```
原始："瞳孔放大，汗珠从额头滴落"
转换："瞳孔放大 汗珠滴落"

原始："手指颤抖，试图敲击键盘但键盘完全失效"
转换："手指颤抖 敲击键盘"

原始："伊娃以蓝色全息投影形式出现，温柔地微笑"
转换："微笑"
```

**动作词汇库**：
- 面部表情：微笑、皱眉、愤怒、恐惧、哭泣、凝视、眨眼
- 身体动作：走动、奔跑、坐下、站立、转身、拥抱、推开
- 手部动作：指、握、拍、握拳、颤抖
- 眼神动作：凝视、闪烁、眨眼、看向

### 4. 构图标准化

**规则**：
- 统一使用标准构图术语
- 移除详细描述，只保留构图类型

**构图对照表**：

| 原始描述 | 标准术语 |
|----------|----------|
| 三分法、黄金分割 | 三分法 |
| 中心对称、对称构图 | 中心对称 |
| 对角线、倾斜构图 | 对角线 |
| 框架式、门框拍摄 | 框架式 |
| 广角、超广角 | 远景 |

**示例**：
```
原始："眼睛放在画面的三分线交叉点上"
转换："三分法"

原始："人物居中，左右对称"
转换："中心对称"

原始："利用前景的门框形成框架效果"
转换："框架式"
```

| 硬光、强烈光线 | 硬光 |
| 蓝光、红色光 | 蓝光/红光 |

**示例**：

## Seedance提示词格式规范

### 标准格式

```
集数:X
镜头:Y
提示词:[角色] [景别] [动作] [构图] [光影] [氛围]
```

### 批量格式

```
集数:1
镜头:1
提示词:男1 大特写 瞳孔放大 三分法 侧光 紧张
---
镜头:2
提示词:全景 全景 警报闪烁 中心对称 红光 危险
---
镜头:3
提示词:男1 特写 手指颤抖 框架式 顶光 焦虑
```

### 格式说明

1. **集数**：使用数字1、2、3
2. **镜头**：使用数字1、2、3...按顺序排列
3. **提示词**：使用空格分隔各个要素
4. **要素顺序**：角色、景别、动作、构图、光影、氛围
5. **多角色**：使用"+"连接，如"男1+女1"
6. **多重氛围**：使用空格分隔，如"危险 悬疑"

## 特殊情况处理

### 1. 无角色镜头

**规则**：
- 使用"全景"、"屏幕"、"文字"等代替角色
- 或直接省略角色

**示例**：
```
原始："整个研究所突然断电，陷入黑暗"
转换："全景 全景 断电 中心对称 黑暗 紧张"
```

### 2. 纯文字镜头

**规则**：
- 使用"文字"作为角色
- 在动作中描述文字内容

**示例**：
```
原始："屏幕出现文字：有时候，最伟大的觉醒..."
转换："文字 特写 文字居中 中心对称 黑色 白色 升华"
```

### 3. 环境镜头

**规则**：
- 使用"全景"、"环境"作为角色
- 强调场景描述

**示例**：
```
原始："废墟与远处的城市对比"
转换："全景 远景 废墟城市 对比 夕阳 希望"
```

### 4. 声音镜头

**规则**：
- 使用"声音"作为角色
- 强调听觉元素

**示例**：
```
原始："黑暗中只有伊娃的声音"
转换："声音 全景 黑暗 中心对称 黑暗 神秘"
```

## 转换流程

### 步骤1：读取分镜表

从分镜表文件中读取所有镜头信息。

### 步骤2：逐镜头转换

按照转换规则，逐个镜头进行转换。

### 步骤3：格式化输出

将转换后的提示词按照Seedance格式输出。

### 步骤4：验证检查

检查转换后的提示词是否符合格式规范。

## 注意事项

1. **保持简洁**：Seedance提示词不宜过长，每镜头控制在10个词以内
2. **关键词优先**：优先使用Seedance能识别的关键词
3. **角色标识**：严格使用角色标识符，不要使用描述
4. **格式统一**：保持批量格式的一致性
5. **可读性**：提示词应易于理解，避免歧义

## 常见问题

**Q: 动作描述太长怎么办？**
A: 只保留核心动作，移除修饰词和细节描述

**Q: 多个氛围词如何处理？**
A: 使用空格分隔，如"危险 悬疑"

**Q: 如何处理特殊效果？**
A: 将效果转化为关键词，如"爆炸" → "爆炸"、"烟雾" → "烟雾"

**Q: 光影描述太复杂怎么办？**
A: 只保留主要光影类型，移除细节描述

**Q: 如何确保转换准确性？**
A: 遵循转换规则，使用标准术语，保持格式一致
