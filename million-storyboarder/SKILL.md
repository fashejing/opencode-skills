# 百万分镜师 - AI短片分镜生成工具

基于AI技术，将用户的简短主题转化为专业的3分钟短片分镜设计，适用于YouTube、TikTok等社交媒体平台。

## 快速开始

### 交互式工作流程

当用户提供一个主题时，系统将引导完成以下步骤：

1. **主题澄清** - 了解故事类型、核心冲突、目标受众
2. **电影风格选择** - 从15+种电影美学风格中选择
3. **影调设定** - 选择明暗对比、色彩方案、光线风格
4. **角色表生成** - 创建故事角色档案
5. **第1集剧本生成** - 1分钟剧情，开篇抓眼球，末尾留悬念
6. **第1集分镜设计** - 结构化分镜提示词
7. **第1集审核** - 用户选择 OK / 重写该集
8. **第2集生成与审核** - 重复步骤5-7
9. **第3集生成与审核** - 重复步骤5-7，结尾升华立意
10. **Seedance提示词转换** - 将分镜转换为文生视频提示词

## 剧本编写规范

### 整体结构

- **总时长**：3分钟，分为3集
- **每集时长**：1分钟（约15-20个镜头）
- **播放顺序**：第1集 → 第2集 → 第3集（连续故事）

### 第1集：开篇抓眼球

**目标**：前3秒完播率最大化

**开篇要求**：
- 第一个镜头必须极具视觉冲击力
- 避免缓慢铺垫，直接进入冲突或高潮
- 使用强烈的视觉元素：爆炸、惊险动作、神秘事件

**结尾要求**：
- 留下巨大悬念，让观众情不自禁想看第2集
- 悬念类型：危机未解、角色命运未知、新谜团出现

### 第2集：矛盾升级

**目标**：维持观众兴趣，深化故事

**开篇要求**：
- 快速承接第1集结尾的悬念
- 节奏紧凑，不断推进剧情

**结尾要求**：
- 留下比第1集更大的悬念
- 悬念类型：更大危机、重要真相即将揭示、关键角色生死未卜

### 第3集：高潮与升华

**目标**：震撼高潮 + 立意升华

**开篇要求**：
- 直接进入高潮阶段，解决第2集的悬念

**结尾要求**：
- 给出震撼性的结局
- 升华立意：人性探讨、社会思考、哲学启示
- 避免庸俗结局，追求深度和回味

### 悬念设计技巧

**视觉悬念**：
- 镜头突然黑屏，听到声音但看不到画面
- 画面边缘出现神秘人物
- 重要物体突然消失或出现

**剧情悬念**：
- 角色说出"你绝对想不到..."
- 时间倒流或异常现象
- 两个看似无关的事件突然关联

**情感悬念**：
- 角色面临道德抉择
- 亲密关系的背叛
- 真相与情感的冲突

## 分镜设计规范

### 分镜结构化提示词模板

每个分镜必须包含以下要素：

```
[角色] [景别] [动作] [构图] [光影] [氛围] [时长]
```

**示例**：
```
男1 特写 眼中闪过一丝恐惧 三分构图 侧光阴郁 紧张 2秒
女1 全景 慢步走向镜头 中心对称 顶光神秘 诡异 3秒
```

### 角色标识系统

使用以下标识符代替具体角色描述：

- **男1** - 男主角
- **女1** - 女主角
- **男2** - 男配角1
- **女2** - 女配角1
- **男3** - 男配角2
- **女3** - 女配角2
- **群演** - 背景群众演员

### 景别分类

| 景别 | 说明 | 用途 | 时长建议 |
|------|------|------|----------|
| 远景 | 展示广阔环境、宏大场景 | 建立场景、氛围营造 | 3-5秒 |
| 全景 | 展示人物全身及周围环境 | 人物关系、空间关系 | 2-4秒 |
| 中景 | 展示人物腰部以上 | 对话、动作展示 | 2-3秒 |
| 近景 | 展示人物胸部以上 | 情感表达、细节 | 1-2秒 |
| 特写 | 展示人物面部或物体细节 | 情绪、重要细节 | 1-2秒 |
| 大特写 | 展示眼睛、嘴唇等极小细节 | 极度紧张、情绪爆发 | 0.5-1秒 |

### 构图原则

**三分法构图**：
- 将画面分成3×3网格
- 主体放在交叉点上
- 适用于大多数场景

**中心对称构图**：
- 主体居中，左右对称
- 传达稳重、神圣、压抑感
- 适合严肃场景

**对角线构图**：
- 主体沿对角线排列
- 增加动感和张力
- 适合动作场景

**黄金分割构图**：
- 主体按黄金比例排列
- 传达美感与和谐
- 适合唯美场景

**框架式构图**：
- 利用前景形成框架
- 增加深度和层次
- 适合侦探、悬疑题材

### 光影风格

| 光影类型 | 特点 | 适用场景 |
|----------|------|----------|
| 侧光 | 明暗对比强烈，立体感强 | 悬疑、惊悚 |
| 顶光 | 压抑、神秘感 | 科幻、犯罪 |
| 底光 | 恐怖、诡异 | 恐怖片 |
| 逆光 | 剪影效果，神秘感 | 浪漫、回忆 |
| 柔光 | 温馨、柔和 | 爱情、家庭 |
| 硬光 | 强烈对比，戏剧性 | 西部、黑帮 |

### 氛围营造

**紧张感**：
- 快速剪辑（0.5-1秒/镜头）
- 手持摄影抖动
- 低频音效

**神秘感**：
- 低对比度，偏蓝调
- 慢镜头
- 空灵音乐

**温馨感**：
- 暖色调（橙、黄）
- 柔光
- 缓慢运动

**恐惧感**：
- 极端对比（黑/白）
- 突然音效
- 闪烁灯光

### 审美标准

**构图要求**：
- 避免画面杂乱
- 保持视觉平衡
- 有明确视觉焦点
- 利用引导线引导视线

**色彩要求**：
- 色彩和谐统一
- 符合影调设定
- 避免过多色彩冲突
- 利用对比色突出主体

**动态要求**：
- 镜头运动要有目的性
- 避免无意义抖动
- 运动速度符合情绪
- 多种运动方式交替

## 用户审核流程

### 审核机制

**逐集审核**：
- 第1集生成完成后，先审核第1集
- 用户可以选择：
  - ✅ **OK** - 确认该集通过
  - 🔄 **重写该集** - 重新生成该集剧本和分镜
- 通过后，才继续生成第2集
- 依次类推，完成全部3集审核

**分镜级别审核**：
- 在逐集审核中，用户可以指定修改特定分镜
- 例如："第1集第5个分镜改成中景，增加紧张感"
- 支持多次迭代优化

### 审核问题

当用户要求重写或修改时，需要明确以下信息：

1. **重写范围**：
   - 重写整集
   - 修改特定分镜（需说明集数和分镜序号）

2. **修改方向**：
   - 增强视觉冲击力
   - 调整节奏（加快/放慢）
   - 改变氛围（更紧张/更神秘/更温馨）
   - 优化悬念设置

3. **具体要求**：
   - 特定镜头的调整
   - 角色动作的修改
   - 光影或构图的变化

## Seedance提示词转换

### 转换规则

**角色标识转换**：
- 将具体角色描述转换为角色标识
- 示例："一个穿黑衣的年轻人" → "男1"

**景别描述标准化**：
- 统一使用标准景别术语
- 示例："镜头拉远展示整个房间" → "全景"

**影调映射**：
- 将详细的影调描述转换为Seedance支持的关键词
- 示例："阴郁的蓝色调，侧光" → "阴郁 蓝调 侧光"

**结构化输出**：
```

提示词:男1 特写 惊恐表情 三分构图 侧光阴郁 紧张
```

## 最佳实践

1. **开篇抓眼球**：第1集第一个镜头必须震撼
2. **悬念设置**：每集末尾都要留钩子
3. **节奏控制**：张弛有度，不要一直紧张或一直平淡
4. **视觉统一**：保持电影风格和影调的一致性
5. **角色鲜明**：每个角色都有独特的视觉标识
6. **审美在线**：每个镜头都要精心设计构图和光影
7. **迭代优化**：善用审核机制，反复打磨

## 常见问题

**Q: 如何提高3秒完播率？**
A: 第1集第一个镜头必须极致视觉冲击，避免缓慢铺垫

**Q: 悬念怎么设计才有效？**
A: 利用信息差、时间差、情感差，让观众产生"接下来会发生什么"的强烈好奇

**Q: 如何保持风格统一？**
A: 严格遵循电影风格选项库的视觉特点，不要随意混搭

**Q: Seedance提示词如何优化？**
A: 保持结构化，使用标准术语，避免过于复杂的描述

**Q: 如果用户不满意某集怎么办？**
A: 先了解不满意的具体原因（节奏/视觉/剧情），针对性重写
